# 📦 Vencord Plugin Installation Guide

## 🇬🇧 English

### Requirements
- Vencord installed (Not by VencordInstaller.exe !)
- Discord desktop app

### Installation

1. Open the Vencord plugins folder:
```
%appdata%/Vencord/src/userplugins/
```

2. Create a new folder (example):
```
MyPlugin
```

3. Inside it, create a file:
```
index.tsx
```

4. Paste the plugin code into `index.tsx`

---

### Reload Discord

Press:
```
CTRL + R
```

---

### Enable the plugin

Go to:
```
Settings → Vencord → Plugins
```

Enable your plugin.

---

## 🇫🇷 Français

### Prérequis
- Vencord installé
- Discord desktop

### Installation

1. Ouvre le dossier des plugins Vencord :
```
%appdata%/Vencord/src/userplugins/
```

2. Crée un dossier (exemple) :
```
MyPlugin
```

3. Dans ce dossier, crée un fichier :
```
index.tsx
```

4. Colle le code du plugin dans `index.tsx`

---

### Redémarrer Discord

Appuie sur :
```
CTRL + R
```

---

### Activer le plugin

Va dans :
```
Paramètres → Vencord → Plugins
```

Active ton plugin.

---

## ⚠️ Important

Make sure **"Enable User Plugins"** is enabled in Vencord settings.
